# Smart Navigation Glasses for the Visually Impaired


AI-powered wearable glasses that combine semantic navigation, real-time obstacle detection, and adaptive multimodal feedback to guide visually impaired users in outdoor and indoor environments.


## Features
- Semantic navigation
- Scene understanding
- Offline fallback
- Haptic and bone-conduction feedback
- Voice conversation interface
- Personal learning
- Crowdsourced hazard data
- Indoor navigation via QR/beacons
